F1 = Toggle room window
F2 = Toggle texture window
END = Next room
HOME = Previous room
PAGE UP = Previous texture
PAGE DOWN = Next texture
ENTER = Toggle room highlighting
INSERT = Reset camera

HOLD Right mouse button + move mouse: Rotate camera
Mouse Scroll : Zoom in/out

CTRL + R = Show 'go to room' box

==== Go To Room ==== 
Enter the room number - press enter to go to the room, escape to cancel.
